Activité 1 – Les mathématiques du temps

Situation

Considérez le problème suivant : Écrivez un programme qui convertit un nombre de secondes en nombre d’heures, de minutes et de secondes. La méthode qui permet de lire le nombre de secondes est déjà codée. On vous propose trois implémentations de l’algorithme de conversion. Lisez les implémentations puis répondez aux questions .

Exemples d’exécution

Entrée : 10 898 Sortie : 3h 1m 38s Entrée : 1796 Sortie : 0h 29m 56s

La méthode LireSeconde permet de lire un entier au clavier : 

///<summary>
/// Demande à la personne utilisatrice du système de saisir un nombre entier de secondes au clavier.
/// Si le nombre est de format invalide ou négatif, alors elle doit recommencer la saisie.
/// <returns>Le nombre de secondes saisies par la personne utilisatrice</returns> 
///</summary>
public static LireSecondes() {
    Console.WriteLine("Saisissez le nombre de secondes (nombre entier) a couvertir.");
    int secondes = -1;
    bool saisieValide = false;

    while(!saisieValide) {
        while(!int.TryParse(Console.ReadLine(), out secondes)) {
            Console.WriteLine("Le nombre saisie ne correspond pas au format entier. Saisissez la valeur de nouveau.");
        }

        if(secondes < 0) {
            Console.WriteLine("Le nombre de seconde doit être plus grand ou égal à 0. Saisissez la valeur de nouveau.");
        } else {
            saisieValide = true;
        }
    }

    return secondes;
}

Voici trois implémentations fonctionnelles de l'algorithme qui permettent d’atteindre ce résultat :

Implémentation 1

public static void  ConvertirSecondes() {
    int secondesEntrees = LireSecondes();

    for(int heures = 0; secondesEntrees >= 3600; secondesEntrees -= 3600) {
        heures++;
    }
    for(int minutes = 0; secondesEntrees >= 60; secondesEntrees -= 60) {
        minutes++;
    }
    Console.WriteLine($"{heures}h {minutes}m {secondesEntrees}s");
}

Implémentation 2

public static void ConvertirSecondes() {
    int secondesEntrees = LireSecondes();
    int heures = secondesEntrees / 3600;        
    int minutes = (secondesEntrees % 3600) / 60
    int secondes = ((secondesEntrees % 3600) / 60) % 60

    Console.WriteLine($"{heures}h {minutes}m {secondes}s");
}

Implémentation 3

public static void ConvertirSecondes() {
    int tempsConvertir = LireSecondes();

    int heures = tempsConvertir / 3600;
    tempsConvertir -= heures * 3600;

    int minutes = tempsConvertir / 60;
    tempsConvertir -= minutes * 60;

    int secondes = tempsConvertir;

    Console.WriteLine($"{heures}h {minutes}m {secondes}s");
}

Questions

  1. Lequel de ces algorithmes est le plus simple à lire pour comprendre son fonctionnement ? Quelles caractéristiques font-elles en sorte qu’il est plus facile à comprendre ?
  2. À votre avis, lequel de ces algorithmes est le plus rapide à l’exécution ? Quelles caractéristiques font-elles en sorte qu’il est plus rapide à s’exécuter ?
  3. Quelle implémentation de l’algorithme est la meilleure ? Énoncez les critères utilisés pour justifier quelle implémentation est la meilleure.
