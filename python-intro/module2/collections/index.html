    Collections de données

    Objectifs de la section

    Reconnaître les situations d'utilisation des différentes collections de données
    Utiliser les collections de données : listes, n-uplet, ensembles et dictionnaires

    Temps requis

    45 minutes

    Utilité des collections de données

    Imaginons que l'on effectue un laboratoire en chimie et que l'on observe la température d'une solution toute les secondes pendant 10 secondes. Pour importer les données dans un programme, il faudrait créer 10 variables et calculer par exemple la plus haute température demanderait l'écriture de 9 conditions (if). Ça serait un travail beaucoup trop long pour être efficace, et encore il n'y a que 10 données !

    Les collections permettent de gérer plusieurs valeurs dans une seule variable et d'accéder dynamiquement au contenu. Elles sont très utilisées en programmation. Dans les prochains modules, nous verrons 6 types de collections fréquemment utilisées :

    • 4 collections natives de Python (liste, n-uplet, ensemble et le dictionnaire)
    • Les tableaux (array) de NumPy
    • Les tableaux de données (dataframe) de Pandas

    Descriptions des collections de données natives

    Liste

    Une liste entrepose séquentiellement des données : les données sont toujours présentées dans le même ordre. On utilise les crochets carrés (square brackets) pour déclarer une liste et l'on place les éléments de la liste dans les crochets carrés en les séparant par des virgules.

    Déclaration d'une liste
    1
    temperatures = [5.0, 5.3, 5.7, 6.0, 5.9, 5.8, 5.5, 5.3, 5.1, 5.1]

    À chaque donnée lui est associé un indice (index) qui permet d'accéder à l'élément dans la liste. Les éléments sont numérotés à partir de 0. On indique entre crochets carrés l'indice de la valeur à laquelle l'on veut accéder. Par exemple, pour accéder à la première valeur (5.0), on indique entre crochets la valeur 0.

    La fonction la plus utile sur les listes est la fonction len qui accepte un paramètre : une liste; puis retourne la longueur de la liste (c'est-à-dire le nombre d'éléments qu'elle contient).

    Fonctions sur les listes numériques

    Pour les listes composées uniquement d'éléments numériques, quelques fonctions existent pour extraire des résumés ou des informations sur la liste. Toutes ces fonctions acceptent qu'un paramètre, la liste pour laquelle obtenir l'information.

    Maximum

    La fonction max retourne la plus grande valeur d'une liste

    Minimum

    La fonction min retourne la plus petite valeur d'une liste

    Somme

    La fonction sum retourne la plus petite valeur d'une liste

    Ajouter un élément à une liste

    Deux techniques existent pour ajouter des éléments à une liste, soit à la fin, soit à une position précise (déterminée par un indice).

    Ajouter un élément à la fin d'une liste

    On peut ajouter un élément à la fin d'une liste avec la méthode append.

    Ajouter un élément à une liste

    Méthode et fonction

    Une méthode est une fonction qui est associée à une classe (concept de programmation orientée objet). La différence à l'usage est la suivante :

    • Fonction : appel sans référence (comme len ou range)
    • Méthode : appel sur une donnée avec le symbole point (comme append)

    Attention aux ajouts

    Il ne faut pas ajouter une liste à une autre avec append, il faut plutôt utiliser la méthode extend ou l'opérateur + décrit à la section Joindre deux listes et répéter une liste.

    Ajouter un élément à une position précise

    Pour ajouter un élément dans une liste à un endroit précis, on utilise la méthode insert qui accepte aussi l'indice où insérer les éléments. Tous les éléments dont l'indice est égal ou supérieur à l'indice inséré seront décalés « d'une case vers la droite » donc verrons leur indice augmenter de 1.

    Il existe deux techniques pour retirer un élément d'une liste : on peut le retirer par sa valeur ou par son indice.

    Joindre deux listes et répéter une liste

    On peut combiner deux listes avec l'opérateur +. Dans ce cas, une nouvelle liste est créée (aucune des deux listes originales n'est modifiée) tous les éléments de la première puis tous les éléments de la seconde.

    On peut créer une nouvelle liste en répétant les valeurs d'une liste avec l'opérateur *. On indique après la liste, puis l'opérateur et le nombre de fois pour lequel répliquer la liste. Une nouvelle liste est créée avec les éléments répétés.

    N-uplet

    Pour traiter des collections de données immuables, il est possible d'utiliser les n-uplets (tuples). On les déclare en utilisant des parenthèses et l'on accède aux éléments de la même façon que pour les listes (toujours des crochets carrés et le système des indices).

    Il est aussi possible d'appeler les fonctions arithmétiques ainsi que les opérateurs + et * sur les n-uplets. Cependant, les éléments d'un n-uplet sont immuables et on ne peut pas ajouter ou retirer des éléments d'un n-uplet.

    Exemple d'utilisation des n-uplets
    1
2
3
    point = (2, 4)      # On déclare un point du plan cartésien
print("Les coordonnées du point sont (" + str(point[0])+ ", " + str(point[1]) + ")")
# Affiche Les coordonnées du point sont (2, 4)

    Ensemble

    Les ensembles sont des collections de données semblables aux listes avec la restriction que chaque élément ne peut apparaître qu'une seule fois. On les déclare en utilisant les accolades { }. Comme les éléments dans un ensemble ne sont pas ordonnés, on ne peut pas leur accéder avec le système des indices.

    Créer un ensemble vide

    Contrairement aux autres conteneurs, on ne peut pas simplement indiquer une paire d'accolades vides pour déclarer un ensemble vide, il faut utiliser la fonction set(). Une paire d'accolades vides créent un dictionnaire.

    Parcourir un ensemble

    Pour énumérer les éléments dans un ensemble, on utilise une boucle. On place à droite de l'opérateur in et la variable « compteur » contiendra plutôt les valeurs successives des éléments de l'ensemble.

for nombre in nombres_premiers:
    print(nombre)

    À propos de la fonction range

    La fonction range retourne une collection de données. Lorsque l'on écrit range(len(liste)) on génère un ensemble de valeurs qui correspondent aux indices de la liste, puis on parcourt cette liste. Ce qui a été présenté plus haut est un raccourci de dire que l'on parcourt les indices de la liste, on parcourt plutôt les valeurs d'indice de la liste.

    Vérifier si une valeur est présente

    Pour déterminer si une valeur se trouve dans un ensemble, on utilise l'opérateur in, mais dans une opération autre qu'une boucle. Le membre de gauche doit aussi être une valeur ou une variable qui contient une valeur. Pour vérifier l'absence d'une valeur dans un ensemble, on utilise l'opérateur not in

    La méthode pop

    La méthode pop existe aussi pour les ensembles, mais retire le plus petit des éléments.

    Opération des ensembles

    Les opérations mathématiques définies sur les ensembles sont disponibles dans le langage Python nativement.

    Opération Opérateur Effet
    Différence A - B Retourne un ensemble des éléments qui sont dans A, mais pas dans B
    Différence symétrique A ^ B Retourne un ensemble des éléments qui sont dans A ou B, mais pas ceux dans A et B
    Est un sous-ensemble A <= B ou A < B True si A que des éléments de B
    Est un super-ensemble A >= B ou A > B True si A contient au moins tous les éléments de B
    Intersection & Retourne un ensemble des éléments qui sont dans A et dans B
    Union | Retourne un ensemble des éléments qui sont dans A ou dans B

    Dictionnaire

    Un dictionnaire permet de stocker des données qui associent des clés et des valeurs (parfois appelé fautivement tableau associatif). Les dictionnaires sont très utiles pour gérer des configurations ou des informations statiques et calculées à partir d'autres valeurs. On déclare le dictionnaire de la façon suivante :

    Déclaration de dictionnaire
    1
    dictionnaire = {cle1 : valeur1, cle2 : valeur2, ...}

    N'importe quel type de donnée peut être utilisé comme valeur dans un dictionnaire. Pour les clés elles doivent être dans l'un des trois types suivants : - Un nombre, - Une chaîne de caractères, ou - Un n-uplet qui contient que des nombres, des chaînes ou d'autres n-uplets composés seulement de nombres ou de chaînes.

    Dans un dictionnaire, chaque clé doit être unique. On peut voir les clés comme un « index » plus général. Cependant, tout comme les ensembles, les dictionnaires ne sont pas triés.

    Bien coder ses dictionnaires

    Bien qu'il soit possible d'utiliser plusieurs types de clés dans un dictionnaire (par exemple des clés entier et chaîne de caractères), le code est plus facile à comprendre si un seul type de clé est utilisé (seulement des chaînes ou seulement des valeurs de clé par exemple).

    Pour accéder à une valeur dans un dictionnaire, on utilise aussi les crochets carrés, mais en indiquant une clé existante entre les croches. Pour vérifier si une clé existe, on utilise le mot clé in comme pour les ensembles.

    Parcourir un dictionnaire

    Il existe deux façons de parcourir un dictionnaire : soit l'on parcourt ses clés ou l'on parcourt les clés et les valeurs en même temps.

    Parcourir selon les clés

    Pour parcourir selon les clés, on utilise la boucle for et la variable d'itération devient la clé. À chaque passage de la boucle, une clé qui n'a pas déjà été parcourue est sélectionnée. Il n'y a pas d'assurance que les éléments sont parcourus dans l'ordre dans lequel ils ont été ajoutés. Par exemple, parcourons un dictionnaire qui pour un symbole chimique retourne le nom complet de l'élément.

    def distance(p0, p1):
    """Calcule la distance entre deux points (longueur d'un segment) et la retourne.

    Keywords arguments:
    p0 -- le premier point du segment.
    p1 -- le second point du segment.
    """
    return sqrt((p0[0]-p1[0])**2 + (p0[1]-p1[1])**2)

def info_triangle(p0, p1, p2):
    """Calcule l'aire, le périmètre et la longueur des côtés d'un triangle et retourne
    un dictionnaire avec les informations.

    Keywords arguments:
    p0 -- le premier point du triangle.
    p1 -- le second point du triangle.
    p2 -- le troisième point du triangle.
    """
    cotes = { "a" : distance(p0, p1), "b" : distance(p0, p2), "c" : distance(p1, p2) }

    # Calcul du perimetre
    perimetre = 0.0                     # Ici sum(cotes) ne fonctionne pas, car c'est un dictionnaire

    for cote in cotes:
        perimetre += cotes[cote]        # Même chose que perimetre = perimetre + longueur

    # Calcule de l'aire
    s = 0.5 * perimetre
    aire = sqrt(s * (s - cotes["a"]) * (s - cotes["b"]) * (s - cotes["c"]))

    return {
        "aire" : aire,
        "cotes" : cotes,
        "perimetre" : perimetre    
    }

# Exécution de la fonction
print(info_triangle((0, 0), (2, 2), (5, -1)))

    Modifier un dictionnaire

    On peut modifier les dictionnaires en changeant leur valeur, en ajoutant une valeur ou en retirant une valeur. Tout comme les autres conteneurs, on ne peut pas retirer de valeurs pendant que l'on itère dans un dictionnaire. Une contrainte supplémentaire s'impose, il n'est pas possible de modifier une clé. Pour changer la clé d'un élément, il faut plutôt dupliquer l'élément en lui affectant la nouvelle clé et supprimer la copie qui est associée à la version de la clé avant la modification.

    Les one-liner de Python

    Python permet d'écrire plusieurs expressions sous la forme de one-liner (instruction qui tient sur une ligne). Si vous remarquez dans un code une expression entre parenthèses, crochets carrés ou accolades avec de courtes instructions (comme des if et des for) à l'intérieur, il y a de fortes chances qui s'agissent de one-liner. Pour les personnes intéressées, on réfère à ses expressions comme des comprehension, par exemple list comprehension.
