    La syntaxe des fonctions

    Les fonctions, d'un point de vue mathématique sont des opérateurs complexes qui acceptent une ou plusieurs variables et retourne une ou plusieurs valeurs. Une fonction mathématique a comme seule contrainte que pour les mêmes valeurs acceptées, le même résultat est retourné (autrement dit chaque valeur du domaine est associée à une seule valeur de l'image de la fonction). On peut penser aux fonctions polynomiales ou trigonométriques classiques (\(f(x) = x\), \(f(x) = 5\) ou \(f(x, y) = x+y\)).

    En informatique, le terme fonction désigne une idée similaire : soit le fait de prendre des valeurs et d'appliquer des opérations sur ces valeurs. Les fonctions sont des blocs de code qui encapsulent un traitement. Ces blocs de code acceptent aucune, une ou plusieurs variable et retourne aucun ou un seul résultat. Dans le code, on retrouve une instruction qui vise à déclarer (declare) une fonction et une instruction qui vise à appeler ou invoquer (call ou invoke) la fonction.

    La déclaration d'une fonction se fait avec le mot-clé def.

    Paramètres

    Pour permettre à la fonction d'accepter des paramètres, on ajoute des identificateurs des paramètres entre les parenthèses dans la définition. Les paramètres agissent comme des variables locales et dont l'initialisation se fait à l'appel de la fonction. Par exemple, on définit la fonction qui calcule l'aire d'un rectangle. Cette fonction aura les paramètres de base et de hauteur.

    aire = base * hauteur

# Invocation pour un rectangle avec une base de 5 unités et une hauteur de 3 unités
aire_rectangle(5, 3)    # Affiche 15

# Invocation pour un rectangle avec une base de 65 unités et une hauteur de 873 unités
aire_rectangle(65, 873) # Affiche 56 745

    Portée des variables et des paramètres

    Les variables déclarées dans les fonction et les paramètres ont une portée scope limitée à la fonction qui les déclare. Autrement dit, les variables et les paramètres suivent ces deux comportements :

    • On peut réutiliser les identificateurs utilisés ailleurs sans risque d'effet de bord (que la modification d'une ait un impact sur l'autre); et
    • On ne peut pas utiliser les valeurs des paramètres et variables déclarées dans la fonction hors de celle-ci.

    Valeur de retour

    Les fonctions peuvent retourner une valeur. On indique la valeur à retourner avec l'instruction return. Les fonctions peuvent avoir plusieurs instructions return, mais dès que la première est rencontrée, alors la fonction s'arrête.

    def aire_rectangle(base, hauteur):
    aire = base * hauteur
    return aire

aire_calculee = aire_rectangle(5, 7)

print("L'air du rectangle est : " + str(aire_calculee))

    La fonction taux_taxes contient plusieurs instructions return. Chacune correspond à une condition d'appel particulière et arrête l'exécution de la fonction dès qu'elle est rencontrée.

    def taux_taxes(taxable_provincial, taxable_federal):
    if taxable_provincial == True and taxable_federal == True:
        return 0.15
    elif taxable_provincial:
        return 0.9975
    elif taxable_federal:
        return 0.05

    return 0.0
