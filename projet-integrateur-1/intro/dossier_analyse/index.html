    Contenus de la section

    4-5 Application des règles de mise en page pour la production de rapports conformes

    Temps requis

    10 minutes

    Dans la remise d'un document d'analyse, certes le contenu et l'organisation sont les principaux facteurs déterminants de la qualité. L'aspect de présentation matériel facilite la compréhension. Voici quelques éléments de présentation importants à respecter.

    Présentation matérielle du Cégep

    Le gabarit d'un document d'analyse et de conception est adapté de la norme IEEE 830 et ISO/IEC/IEEE 29148:{2011,2017}. Cette norme est difficilement compatible avec le standard Dionne généralement utilisé dans la présentation des autres travaux. Il est donc normal que votre document ne ressemble pas au standard habituellement exigé dans les cours du Cégep.

    Format du texte

    Le texte doit être présenté dans une police, fonte et taille identique dans le document. Tout changement attire l'attention lors de la lecture. Donc si le changement est involontaire ou par négligence, vous risquez d'inférer un faux raisonnement chez la personne qui lit votre analyse.

    Pour attirer l'attention sur des éléments, on utilise la fonte grasse. Les fontes italiques sont réservées pour les termes anglais, alors que le soulignement est réservé pour les références.

    Il est d'usage d'utiliser une police neutre, sans empattement comme Arial, Calibri ou Aptos avec une taille entre 10 et 12 points, selon la police utilisée. L'écriture est systématiquement noire sur page blanche.

    Numérotation des éléments

    Les titres doivent être numérotés, de même que les pages de votre document. Utilisez la fonction de renvoi de Word pour faire référence à un élément numéroté, il se mettra à jour automatiquement selon les changements dans l'ordonnancement que vous allez faire dans les révisions de votre document.

    Généralement pour les titres on se limite à numéroter les trois premiers niveaux de titre. On ne numérote pas les tableaux ou les figures auxquelles on ne fait jamais référence (dont celle dans les descriptions longues de CU particulièrement).

    Références entre les sections
    On ne doit jamais dupliquer des informations dans un document d'analyse. On utilise donc les références entre les sections et les contenus abondamment. Il est de bonne pratique de souligner le titre des sections ou des éléments lorsqu'il s'agit d'une référence. Cela permet de distinguer rapidement les références. Évidemment, on utilise le titre exact de l'élément référé.
